# job-hunting-bot-n8n
Automated LinkedIn job alert workflow built with n8n. Uses Apify to fetch real LinkedIn jobs with native filters, processes jobs safely with rate limits, applies AI-based relevance scoring, stores results in Google Sheets, and sends shortlisted job alerts to Telegram without fragile HTML scraping.
